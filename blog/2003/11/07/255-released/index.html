2.5.5 Released!

Nov 7th, 2003

Changes (see ChangeLog for details): * FreeBSD users will be happy to know that all standard metrics are now reported on FreeBSD thanks to the work of Brooks Davis. * This release also fixes a bug brought to our attention by Jim Prewett. This bug in gmond allows a specially crafted UDP packet to crash all gmond on a multicast channel with a segfault. This special UDP packet cannot be created using gmetric but requires a custom piece of code or a fuzzer. * The ganglia documentation is now included in the distribution as ganglia.html and README. We are working to make sure the documentation is simpler to use, up-to-date and directly answers the questions that you have about ganglia. Thank you for using Ganglia!
